# CONTEXT_SPEC.md — 背景脈絡層規格

> 目的：讓讀者理解經典發生時的**世界背景**——時代、政治地理、社會、思想環境、在典記載的天災人禍。
> **不是**「教義為何這樣判」（那是高爭議層，本規格不碰）。背景是背景，不是教義的因果解釋。
> 出處紀律延續 L1/L2，並細化為四級（§3）。

---

## 0. 範圍界定

**做**：時代的政治地理、社會經濟、思想潮流、經典中記載的飢荒/瘟疫/戰爭、季風與雨安居這類環境事實。
**不做**：細粒度氣候災害重建、把特定經繫於特定災年、用背景「解釋」教義為何如此（除非經文明載）。

---

## 1. 三種背景內容，掛在不同層級

關鍵：背景**多半不是 per-segment**，硬塞進逐句會干擾讀經。依其自然粒度掛載：

| 模組 | 粒度 | 是什麼 | 主來源 |
|---|---|---|---|
| A 此經緣起 | per-sutta | 此經在何地、對誰、因何事而說 | **經文本身**（nidāna，L1） |
| B 人地事詞條 | per-entity | 人物、地點、事件 | DPPN（公共領域） |
| C 時代與世界背景 | **per-period / per-place** | 政治、社會、思想、天災人禍 | 學界 + 在典事件 |

A 已在經文中、B 跟字典頁同模式，**C 是你要的新東西**，以下細談。

---

## 2. 時代背景模組（C）

掛在「時代」與「地點」，不掛在 segment。內容分四束：

### 2.1 政治地理
- 十六大國（Mahājanapadas）、第二次城市化的恆河平原。
- 主要王國與都城：摩竭陀（王舍城；頻婆娑羅→阿闍世）、憍薩羅（舍衛城；波斯匿）。
- 共和/寡頭：跋耆/離車（毗舍離）、釋迦族（迦毗羅衛）、末羅。
- 動態：摩竭陀的擴張與兼併。

### 2.2 社會經濟
- 城鎮、貿易、貨幣興起；居士/商人（gahapati）階層抬頭。
- 婆羅門正統 vs 新興沙門（出家遊行）潮流。

### 2.3 思想環境
- 沙門運動；同代的耆那教（大雄 Mahāvīra）、六師外道等並存的思想市場。
- *呈現原則：客觀並陳，不以佛教立場貶抑他派（教義中立同樣適用於對手）。*

### 2.4 天災人禍
- **在典記載者（L1，價值最高）**：如《大般涅槃經》(DN16) 開篇阿闍世欲伐跋耆；琉璃王(Viḍūḍabha)滅釋迦族；典籍中的飢荒、瘟疫記載。
- **環境事實（L1，安全）**：季風週期與「雨安居」——僧團於雨季定居,是環境直接形塑修行制度的乾淨例子。
- **學界重建（標明爭議）**：更廣的災荒/戰亂背景,具名引用,不寫死年代。

## 3. 出處四級標記（延伸 L1/L2）

背景層每一條事實**必帶**一個出處級別,UI 據此標示、可追溯:

| 級別 | 意義 | 視覺 |
|---|---|---|
| 正典 | 經文明載 | 暖墨,可跳該經段 |
| 註釋傳統 | 出自註釋書/後世傳述 | 標「註釋傳統」,與正典區隔 |
| 學界 | 現代學術重建,**具名引用** | 標來源+「學界觀點,有不同說法」 |
| AI | 模型生成的摘要/串接 | 冷灰 AI 徽章,grounded on 上述 |

> 這把「這句話憑什麼這麼說」變成永遠可查。第 3/4 級尤其不可由 AI 自行斷言。

---

## 4. 誠實鐵則（本層最易出錯處）

1. **絕對年代不寫死**:連佛陀生卒都無定論。用「傳統繫於」「學界多繫於」,不給假精確。
2. **不做假因果**:除非經文明載,不可寫「因某災故說此經」。背景≠教義成因。
3. **在典事件 與 後世/學界重建 分開標**:不可把註釋傳說當史實。
4. **天災只收可靠者**:季風/雨安居這種在典環境事實可收;細粒度氣候災害重建不做。
5. **他派客觀並陳**:描述耆那、外道等,不以教義立場褒貶。

## 5. 呈現方式(不干擾讀經)

- **「此經背景」面板**(A):研經頁可摺疊側欄,顯示地點/聽眾/緣由,連到相關詞條。預設摺疊。
- **地點詞條的歷史欄**(B+C):點地名 → 詞條附該地的時代背景(如「王舍城:摩竭陀都城,頻婆娑羅在位」)。
- **「早期佛典的世界」總覽頁**(C,`/world`):獨立頁,per-collection,放政治地理/社會/思想/天災人禍的通論。**不塞進逐 segment。**

### 5.1 世界總覽頁結構(J-5)
四束內容(對應 §2),每束由帶出處的 `SourcedFact` 組成,皆依 SOURCING_STANDARD:
1. **政治地理**:十六大國、摩竭陀/憍薩羅、王與共和邦。經內名單(如增支部十六大國)標 `canonical` 連經段;絕對年代寫範圍。
2. **社會經濟**:第二次城市化、商人/居士階層、貨幣。`scholarly` 重建,年代「約」。
3. **思想環境**:沙門運動、耆那、六師外道並存。客觀並陳,不貶他派。
4. **天災人禍**:在典事件(`canonical`,連經段)/ 環境事實(季風·雨安居)/ 學界重建(標爭議)。
- 頁內設「**年代問題**」說明區:把佛陀年代等無共識議題以各立場具名並列(SOURCING §5A),明標無定論。
- 地點/人物名詞 → 連 B 實體詞條;事件 → 連相關經。
- 材質:`canonical` 暖墨、`scholarly` 冷藍灰附來源、`ai` 冷灰摘要(design-tokens `--prov-*`)。

## 7. V1 / V2 範圍（更新：背景層整體納入 V1）

**V1**:
- A 此經緣起(L1)。
- B 地點/人物詞條的基本歷史(DPPN)。
- 2.4 在典天災人禍事件 + 環境事實(季風/雨安居)。
- **C 世界總覽頁(`/world`,§5.1)** + **學界來源與爭議標注**,一律依 SOURCING_STANDARD:具名引用、絕對年代不寫死、爭議並陳不裁決。

**V2**:思想環境的更深掘、更廣的學界重建擴充、英文化。

> 納入 C 的前提是 SOURCING_STANDARD 把風險關住:無共識者並列具名、年代用範圍、AI 反查對回來源。內容備料(真實學術引用蒐集)為持續工作,非一次到位。

## 8. 風險

| 風險 | 緩解 |
|---|---|
| AI 捏造年代/災害 | 第3/4級不可AI斷言;絕對年代用「傳統/學界繫於」 |
| 背景被當教義成因 | §4 鐵則2:無經證不寫因果 |
| 註釋傳說當史實 | 四級標記強制分流 |
| 對他派失中立 | §4 鐵則5:客觀並陳 |
| 干擾讀經 | 背景進面板/總覽頁,不入逐 segment |
